# Daap - Star Notary

This Daap is part of my work building a solid Blockchain development skill.

## How it works

You can:

- Create a new Star
- Put up the Star for sale
- Transfer the Star to other account
- Exchange the Star
- Check the Star information
- Buy Stars

### Tech used

- Solidity
- Openzeppelin
- Truffle
- ReactJS
- Web3.js
